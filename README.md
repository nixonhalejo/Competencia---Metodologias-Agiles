## 🏭 ¿Qué es ASTERON?

ASTERON es un sistema web de gestión de producción desarrollado para **Macromet S.A.S.**, empresa colombiana de fabricación y exhibición metálica. El sistema digitaliza y centraliza el control de todos los procesos productivos de la planta, eliminando el uso de hojas físicas y registros manuales.

**Tecnologías:** Java · Spring Boot · Spring Security · MySQL · Node.js · Express · Railway

## 🗂️ Módulos del Sistema ASTERON

| Módulo | Descripción |
|--------|-------------|
| Proyectos | Registro y seguimiento de proyectos de producción |
| Clientes y Pedidos | Gestión de clientes, pedidos y detalles de cada orden |
| Fases de Producción | Control de etapas del proceso productivo |
| Actividades | Registro de actividades de cada trabajador |
| Programación de Planta | Planificación de operarios y maquinaria |
| Maquinaria | Inventario y estado de equipos |
| Mantenimientos | Registro de mantenimientos preventivos y correctivos |
| Usuarios y Roles | Control de acceso según el rol de cada persona |
